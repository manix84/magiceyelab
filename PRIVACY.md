# 🔒 Privacy Policy

MagicEyeLab is designed to be privacy-friendly by default.

---

# 📊 Data Collection

MagicEyeLab does not collect personal information.

The application does not:

- 🚫 Require user accounts
- 🚫 Use advertisements
- 🚫 Track user behaviour
- 🚫 Sell data
- 🚫 Use analytics platforms
- 🚫 Upload your images to external servers

---

# 💻 Local Processing

Image generation, depth painting, and pattern creation are processed locally in your browser whenever possible.

📁 Imported files remain on your device unless you explicitly choose to share or export them yourself.

---

# 💾 Local Storage

MagicEyeLab may optionally store small amounts of data in your browser for convenience purposes, such as:

- ⚙️ Saved settings
- 🗂️ Restored generator workspace state
- 🧩 Pattern Maker drafts and tool settings
- 🎨 Depth Painter tool settings
- 🖼️ Imported depth and pattern images, stored locally in a bounded browser-friendly format
- 🎨 UI preferences
- 🧠 Temporary working state

This data never leaves your device.

You can clear this data at any time using your browser's site data/storage controls.

---

# 🌐 Third-Party Services

MagicEyeLab may be hosted using GitHub Pages or similar static hosting providers.

Those services may collect standard technical information such as:

- 🌍 IP address
- 🖥️ Browser type
- 📜 Request logs

Please refer to their respective privacy policies for details.

---

# 👶 Children's Privacy

MagicEyeLab is intended for general audiences and does not knowingly collect information from children.

---

# 🔄 Changes

This privacy policy may be updated over time as the project evolves.

Any significant changes will be documented in the project changelog.
